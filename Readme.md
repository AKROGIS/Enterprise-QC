Enterprise Database QC Tools
============================

This repo contains mainly the tools and instructions for doing Quality
Control on the Alaska Region enterprise geospatial databases. These
databases are in esri Geodatabases using the esri Spatial Database
Engine (SDE) model.

There are other database related tools in this repo as described below.

## FMSSExport

This folder contains instructions and SQL scripts for exporting
data from the Facility Maintenence and Management System (FMSS)
and import the data into our enterprise datasets.  In many cases
our database contains spatial attributes of assets that are tracked
in more detail in the FMSS database.
